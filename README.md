# PEACOCK Renewal

기존 PEACOCK 웹사이트의 식품 중심 콘셉트와 콘텐츠 구조를 유지하면서 반응형·접근성·상호작용을 현대적으로 재구축한 비공식 리뉴얼 프로젝트입니다.

## 실행

```bash
npm install
npm run dev
```

프로덕션 빌드는 `npm run build`로 생성합니다.

## 페이지

- Home
- Brand Story
- Best
- Products
- Gourmet

GitHub Pages에서도 새로고침 오류 없이 동작하도록 hash 기반 경로를 사용합니다.

## 구현 내용

- React 19 + Vite
- Pretendard Variable 단일 폰트
- Phosphor Icons
- 390 / 768 / 1024 / 1440px 반응형 레이아웃
- 제품 검색, 카테고리 필터, 모바일 전체 메뉴, 관심 제품 담기 상태
- WebP 이미지 최적화와 lazy loading

## 자산 안내

이 저장소의 PEACOCK 명칭·제품 이미지·문구는 원본 사이트 리뉴얼 작업을 위한 자료입니다. 공개·상업적 배포 전 해당 브랜드 자산의 사용 및 재배포 권한을 확인해야 합니다.
